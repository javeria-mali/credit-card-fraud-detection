This project is a Machine Learning-based Credit Card Fraud Detection System. It uses a Random Forest model along with SMOTE to handle class imbalance and detect fraudulent transactions.

⚙️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Imbalanced-learn (SMOTE)
Streamlit
Joblib
📊 Features
Data preprocessing and scaling
Handling imbalanced dataset using SMOTE
Random Forest classification model
Real-time fraud prediction
Interactive Streamlit web interface
EDA visualizations (class distribution, confusion matrix)
📁 Project Files
app.py → Streamlit frontend
backend.py → Data processing / model logic
fraud_model.pkl → Trained ML model
dataset.csv → Dataset used
EDA.ipynb → Exploratory Data Analysis
requirements.txt → Dependencies
screenshots/ → Output images
🚀 How to Run Project
pip install -r requirements.txt
streamlit run app.py
📊 EDA Included
Class distribution graph
Confusion matrix heatmap
Data insights visualization
🧾 Conclusion

This project demonstrates how machine learning can be used in financial systems to detect fraudulent transactions. The model provides accurate predictions and helps improve security in banking systems by identifying suspicious activities in real-time.
