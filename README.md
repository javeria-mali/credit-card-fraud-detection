This project demonstrates a machine learning-based Credit Card Fraud Detection System using Random Forest and SMOTE.

The dataset was highly imbalanced, so SMOTE was applied to balance the classes.

After preprocessing, scaling, and training, the model achieved good performance in detecting fraudulent transactions.

A Streamlit web application was built to allow real-time predictions.

Overall, this project shows how machine learning can be applied to financial fraud detection systems to improve security and reduce risk.
